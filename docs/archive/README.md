# Archive Index

Historical notes, superseded implementation writeups, and one-off validation records live here.

## What Belongs Here

- completed rollout summaries that no longer describe the supported repo shape
- implementation notes for removed script-era analytics surfaces
- historical validation reports and migration records
- summaries of retired generated artifact trees when the artifacts themselves are no longer tracked

## What Does Not

- current onboarding or operational guidance
- commands people should still copy into active workflows
- canonical architecture or testing instructions

## How To Use It

- Treat these files as background context, not active source-of-truth documentation.
- If an archive note conflicts with [README.md](../../README.md), [docs/README.md](../README.md), or [docs/guides/usage-guide.md](../guides/usage-guide.md), follow the active docs.
- When retiring an active guide or plan, move the historical writeup here and leave the operational replacement in the active docs path.
- Older archive notes may reference deleted analytics scripts or removed `data/analysis/` and `data/analytics/` trees; preserve those references as history, not as runnable instructions.
