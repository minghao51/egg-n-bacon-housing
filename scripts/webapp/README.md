# Webapp Scripts

Scripts that generate or transform JSON assets consumed by the frontend dashboard.

## Preferred Entrypoints

- `scripts/webapp/prepare_analytics_json.py`
- `scripts/webapp/prepare_interactive_tools_data.py`
- `scripts/webapp/generate_segments_data.py`
- `scripts/webapp/transform_spatial_hotspots.py`

Legacy wrappers remain at `scripts/*.py` for compatibility.
